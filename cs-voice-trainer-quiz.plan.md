# CS Voice Trainer - AI 이론 학습 및 평가 기능 플랜

## 목표
신입사원 교육용 학습자료(Zip)를 바탕으로 AI가 즉석에서 시험 문제를 출제하고 채점/피드백을 제공하는 기능 구현.

## 주요 기능

### 1. 학습 모드 선택
- 메인 화면에서 [실전 음성 시뮬레이션]과 [AI 이론 평가] 모드 선택 가능

### 2. 자료 입력
- **파일 업로드**: 사용자가 로컬에 있는 Zip 파일 업로드
- **자료 처리**: 브라우저에서 Zip 압축 해제 및 텍스트 추출 (JSZip 활용)
- **지원 포맷**: 텍스트 파일(.txt, .md, .json), 코드 파일 등 텍스트로 읽을 수 있는 파일

### 3. AI 문제 출제
- **모델**: Google Gemini 1.5 Flash (긴 컨텍스트 처리 용이)
- **형식**: 객관식(4지선다) 5문항 + O/X 5문항 (총 10문항 세트)
- **내용**: 업로드된 자료의 내용을 바탕으로 핵심 내용 추출 및 문제화

### 4. 시험 응시 및 평가
- 인터랙티브한 문제 풀이 UI
- 제출 시 자동 채점 (생성된 정답지 기반)
- 틀린 문제에 대한 해설 제공

### 5. 게임화 연동
- 시험 점수를 사용자 총점에 반영 (예: 100점 만점 -> 100점 추가)
- 퀴즈 관련 새 배지 추가 (예: "이론 마스터")

## 구현 스텝

1. **환경 설정**
   - `jszip` 패키지 설치
   - `Quiz` 관련 타입 정의

2. **서비스 로직 구현** (`src/services/quiz.ts`)
   - Zip 파일 파싱 및 텍스트 추출 함수
   - Gemini API 연동: 학습자료 -> 퀴즈 JSON 생성 프롬프트

3. **UI 컴포넌트 구현**
   - `ModeSelector`: 메인 화면에서 모드 선택 (시뮬레이션 vs 퀴즈)
   - `QuizHome`: 파일 업로드 및 시작 화면
   - `QuizSolver`: 문제 풀이 화면
   - `QuizResult`: 결과 및 해설 화면

4. **통합**
   - `App.tsx`에 퀴즈 모드 상태 추가 및 화면 전환 처리
   - 퀴즈 완료 시 `userStats` 점수 업데이트
