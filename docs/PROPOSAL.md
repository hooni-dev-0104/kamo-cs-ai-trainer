# CS AI 트레이너 - 제품 기획서

## 1. 문제 정의

### 1.1 기존 고객 서비스 상담원 교육의 문제점

#### 이론 중심 교육의 한계
- **현실**: 대부분의 상담원 교육이 매뉴얼 암기와 이론 강의에 집중
- **문제점**: 
  - 실제 고객과의 대화 경험 부족
  - 다양한 상황에 대한 대응 능력 미흡
  - 이론과 실전의 괴리로 인한 교육 효과 저하
- **영향**: 신입 상담원의 실전 적응 기간이 길어지고, 고객 만족도 하락

#### 실전 교육의 비용 및 시간 문제
- **현실**: 실전 교육을 위해서는 시니어 상담원의 멘토링이나 롤플레이가 필요
- **문제점**:
  - 시니어 상담원의 시간 투입으로 인한 업무 부담 증가
  - 일대일 교육으로 인한 확장성 부족
  - 교육 시간과 비용의 지속적 증가
- **영향**: 교육 예산 증가, 교육 대상자 수 제한

#### 개인화된 피드백 부재
- **현실**: 기존 교육 방식은 일괄적이고 획일적인 피드백 제공
- **문제점**:
  - 개인의 강점과 약점 파악 어려움
  - 맞춤형 개선 방안 제시 불가
  - 학습 진행 상황 추적 어려움
- **영향**: 개인별 역량 향상 속도 저하, 교육 효과 측정 어려움

#### 학습 자료 기반 평가의 한계
- **현실**: 학습 자료를 통한 이론 평가는 수동으로 문제를 출제하고 채점
- **문제점**:
  - 문제 출제에 시간과 인력 소요
  - 학습 자료가 변경될 때마다 문제 재작성 필요
  - 채점 및 피드백 제공의 지연
- **영향**: 교육 담당자의 업무 부담 증가, 학습자 피드백 지연

### 1.2 정량적 문제 정의

- **교육 시간**: 신입 상담원 실전 적응까지 평균 **4-6주** 소요
- **교육 비용**: 상담원 1명당 교육 비용 **월 200만원 이상** (멘토링 시간 포함)
- **고객 만족도**: 신입 상담원 담당 고객 만족도가 기존 상담원 대비 **20-30% 낮음**
- **문제 출제 시간**: 학습 자료 1개당 문제 출제에 **2-3시간** 소요
- **피드백 지연**: 롤플레이 후 피드백 제공까지 **1-2일** 소요

## 2. 해결 방안: CS AI 트레이너

### 2.1 핵심 가치 제안

CS AI 트레이너는 **AI 기반 실전 음성 시뮬레이션**과 **자동화된 이론 평가**를 통해 상담원 교육의 모든 문제를 해결하는 통합 플랫폼입니다.

#### 실전 음성 시뮬레이션으로 경험 축적
- **AI 고객과의 실제 대화**: Google Cloud Text-to-Speech를 활용한 자연스러운 고객 음성 재생
- **감정 표현 포함**: 화난 고객, 답답한 고객 등 다양한 감정 상태 시뮬레이션
- **대화형 AI**: Gemini API를 활용하여 상담원의 응대에 따라 고객의 반응이 동적으로 변화
- **무제한 연습**: 언제든지 반복 연습 가능하여 실전 경험 축적

#### AI 기반 즉시 피드백
- **실시간 분석**: 대화 종료 후 즉시 AI가 응대 내용을 분석
- **다차원 평가**: 공감 능력, 문제 해결 능력, 전문성, 어조 등 4가지 항목 평가
- **개선 제안**: 구체적이고 실행 가능한 개선 방안 제시
- **점수 및 레벨 시스템**: 객관적 성장 지표 제공

#### 학습 자료 기반 자동 평가
- **AI 문제 출제**: 학습 자료(Zip 파일, PPTX 포함)를 업로드하면 Gemini AI가 자동으로 문제 출제
- **즉시 채점**: 문제 풀이 완료 시 즉시 채점 및 상세 해설 제공
- **학습 효과 극대화**: 틀린 문제에 대한 해설을 통해 학습 효과 향상

### 2.2 기대 효과 (정량적)

- **교육 시간 단축**: 실전 적응 기간 **4-6주 → 2-3주**로 단축 (50% 감소)
- **교육 비용 절감**: 멘토링 시간 감소로 상담원 1명당 교육 비용 **월 200만원 → 50만원** (75% 절감)
- **고객 만족도 향상**: 신입 상담원의 실전 경험 축적으로 고객 만족도 **20-30% → 5-10%** 격차로 감소
- **문제 출제 시간 절약**: 학습 자료 1개당 문제 출제 시간 **2-3시간 → 5분** (95% 절감)
- **피드백 즉시 제공**: 롤플레이 후 피드백 제공 시간 **1-2일 → 즉시** (100% 개선)

## 4. 핵심 기능 요약

### 4.1 실전 음성 트레이닝 모듈
1. **시나리오 선택**: 다양한 고객 상담 시나리오 중 선택 (택시 경로 이탈, 잘못된 도착지, 대리기사 지각 등)
2. **AI 고객 음성**: Google Cloud Text-to-Speech를 활용한 자연스러운 고객 음성 재생 (감정 표현 포함)
3. **음성 녹음**: 브라우저 MediaRecorder API를 활용한 응대 내용 녹음
4. **음성 인식**: Google Cloud Speech-to-Text API를 활용한 음성-텍스트 변환
5. **대화형 AI 고객**: Gemini API를 활용한 동적 고객 응답 생성 (상담원의 응대에 따라 감정 변화)
6. **AI 피드백**: Google Gemini API를 활용한 응대 내용 분석 및 피드백 제공 (공감, 문제 해결, 전문성, 어조 평가)

### 4.2 AI 이론 평가 모듈
1. **학습 자료 관리**: 관리자가 Zip 파일(텍스트, PPTX 포함) 업로드
2. **자동 문제 출제**: Gemini AI가 학습 자료를 분석하여 객관식 5문제 + O/X 5문제 자동 생성
3. **채점 및 해설**: 즉시 채점 및 상세 해설 제공
4. **점수 반영**: 퀴즈 점수가 사용자 통계에 반영되어 레벨업 및 리더보드 경쟁

### 4.3 관리자 시스템 모듈
1. **사용자 권한 관리**: 관리자가 다른 사용자의 권한을 지정/해제
2. **학습 자료 관리**: 자료 업로드, 삭제 기능
3. **통계 모니터링**: 전체 사용자 통계 확인 (선택 사항)

### 4.4 게임화 모듈
1. **레벨 시스템**: 점수에 따른 레벨업 (100점당 1레벨)
2. **배지 시스템**: 다양한 업적 달성 시 배지 획득 (첫 걸음, 완벽주의자, 열정가, 마스터 등)
3. **리더보드**: 전체 사용자 순위 경쟁
