- **비밀번호**: `demo123456`

## 5분 데모 시나리오 (빠른 소개)

### 목표
CS AI 트레이너의 핵심 기능을 빠르게 소개

### 단계별 진행

#### 1단계: 로그인 및 모드 선택 (30초)
1. 로그인 화면에서 `demo@example.com` / `demo123456`로 로그인
2. 모드 선택 화면 확인
   - **하이라이트**: "실전 음성 트레이닝"과 "AI 이론 평가" 두 가지 모드 제공

#### 2단계: 실전 음성 트레이닝 체험 (2분)
1. "실전 음성 트레이닝" 선택
2. 시나리오 선택 화면에서 "택시 경로 이탈 불만" 선택
   - **하이라이트**: 다양한 시나리오 제공 (택시, 대리기사 관련)
3. AI 고객 음성 듣기
   - **하이라이트**: 자연스러운 한국어 음성, 감정 표현 포함
4. 음성 녹음 (간단히 "안녕하세요, 불편을 드려 죄송합니다" 정도)
   - **하이라이트**: 브라우저에서 직접 녹음, 음성 인식
5. AI 고객 응답 확인
   - **하이라이트**: 상담원의 응대에 따라 고객 감정이 동적으로 변화
6. "대화 종료" 버튼 클릭
7. 피드백 화면 확인
   - **하이라이트**: 4가지 항목 평가 (공감, 문제 해결, 전문성, 어조), 구체적인 개선 제안

#### 3단계: AI 이론 평가 체험 (2분)
1. "모드 선택"으로 돌아가기
2. "AI 이론 마스터" 선택
3. 학습 자료 목록 확인
   - **하이라이트**: 관리자가 업로드한 학습 자료 목록
4. 자료 선택 및 퀴즈 생성
   - **하이라이트**: AI가 자동으로 문제 출제 (약 10-20초 소요)
5. 퀴즈 풀이 (1-2문제만)
   - **하이라이트**: 객관식 및 O/X 문제, 직관적인 UI
6. 결과 화면 확인
   - **하이라이트**: 즉시 채점, 상세 해설 제공

#### 4단계: 게임화 요소 확인 (30초)
1. 프로필 화면 확인
   - **하이라이트**: 레벨, 총 점수, 완료 세션 수
2. 리더보드 확인
   - **하이라이트**: 전체 사용자 순위 경쟁

## 주요 기능 하이라이트 포인트

### 1. 실전 음성 트레이닝
- **AI 고객 음성**: Google Cloud Text-to-Speech를 활용한 자연스러운 한국어 음성
- **감정 표현**: 화난, 답답한, 슬픈 등 다양한 감정 상태 시뮬레이션
- **대화형 AI**: 상담원의 응대에 따라 고객의 감정과 반응이 동적으로 변화
- **즉시 피드백**: 대화 종료 후 즉시 AI가 4가지 항목 평가 및 개선 제안

### 2. AI 이론 평가
- **자동 문제 출제**: 학습 자료를 업로드하면 AI가 자동으로 문제 출제
- **다양한 형식**: 객관식 5문제 + O/X 5문제
- **즉시 채점**: 문제 풀이 완료 시 즉시 채점 및 상세 해설 제공
- **학습 효과**: 틀린 문제에 대한 해설을 통해 학습 효과 극대화

### 3. 관리자 시스템
- **사용자 권한 관리**: 관리자가 다른 사용자의 권한을 지정/해제
- **학습 자료 관리**: Zip 파일 업로드, 삭제 기능
- **통계 모니터링**: 전체 사용자 통계 확인

### 4. 게임화
- **레벨 시스템**: 점수에 따른 레벨업 (100점당 1레벨)
- **배지 시스템**: 8가지 배지 (첫 걸음, 완벽주의자, 열정가, 마스터 등)
- **리더보드**: 전체 사용자 순위 경쟁

## 예상 질문 및 답변

### Q1: 실제 고객과의 대화와 얼마나 유사한가요?
**A**: AI 고객은 실제 고객과 매우 유사한 경험을 제공합니다. 자연스러운 한국어 음성, 다양한 감정 표현, 그리고 상담원의 응대에 따라 동적으로 변화하는 반응을 통해 실전과 유사한 상황을 시뮬레이션합니다.

### Q2: AI 피드백의 정확도는 어느 정도인가요?
**A**: Google Gemini API를 활용하여 4가지 항목(공감, 문제 해결, 전문성, 어조)을 평가하며, 구체적인 강점과 개선점을 제시합니다. 실제 교육 전문가의 피드백과 유사한 수준의 평가를 제공합니다.

### Q3: 학습 자료는 어떤 형식을 지원하나요?
**A**: Zip 파일 형식을 지원하며, 내부에 텍스트 파일(.txt, .md 등)이나 파워포인트 파일(.pptx)을 포함할 수 있습니다. AI가 자동으로 텍스트를 추출하여 문제를 출제합니다.

### Q4: 교육 비용은 얼마나 절감할 수 있나요?
**A**: 기존 교육 방식 대비 약 75%의 비용 절감이 가능합니다. 멘토링 시간이 크게 감소하고, 자동화된 문제 출제 및 평가로 교육 담당자의 업무 부담이 줄어듭니다.

### Q5: 확장성은 어떤가요?
**A**: Google Cloud API와 Supabase를 활용하여 확장 가능한 아키텍처를 갖추고 있습니다. 사용자 수가 증가해도 안정적으로 서비스를 제공할 수 있습니다.

### Q6: 모바일에서도 사용 가능한가요?
**A**: 현재는 웹 브라우저에서 사용 가능하며, 반응형 디자인으로 모바일에서도 사용할 수 있습니다. 향후 모바일 앱 출시를 계획하고 있습니다.

### Q7: 데이터 보안은 어떻게 관리하나요?
**A**: Supabase의 Row Level Security (RLS) 정책을 통해 사용자별 데이터 접근을 제어하며, 모든 데이터는 암호화되어 저장됩니다.

### Q8: API 비용은 얼마나 드나요?
**A**: Google Cloud API의 무료 티어를 활용하여 초기 비용 부담이 최소화됩니다. 프로토타입 단계에서는 무료 티어로 충분히 운영 가능합니다.
