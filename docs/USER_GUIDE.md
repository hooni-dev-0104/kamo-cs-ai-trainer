## 시작하기

### 1. 회원가입

1. 메인 화면에서 "회원가입" 탭 선택
2. 다음 정보 입력:
   - **이메일**: 유효한 이메일 주소 입력 (예: `user@example.com`)
   - **비밀번호**: 최소 6자 이상
   - **이름**: 선택 사항
3. "회원가입" 버튼 클릭
4. 이메일 인증 (필요한 경우)
   - 이메일로 전송된 인증 링크 클릭
   - 인증 완료 후 자동 로그인

### 2. 로그인

1. 메인 화면에서 "로그인" 탭 선택
2. 이메일과 비밀번호 입력
3. "로그인" 버튼 클릭

### 3. 모드 선택

로그인 후 두 가지 모드를 선택할 수 있습니다:
- **실전 음성 트레이닝**: AI 고객과 음성 대화를 통한 실전 연습
- **AI 이론 마스터**: 학습 자료 기반 AI 퀴즈로 이론 지식 점검

## 실전 음성 트레이닝 사용법

### 1. 시나리오 선택

1. "실전 음성 트레이닝" 모드 선택
2. 시나리오 선택 화면에서 원하는 시나리오 선택
   - **택시 경로 이탈 불만**: 택시 기사가 경로를 이탈한 경우
   - **택시 잘못된 도착지 하차 불만**: 잘못된 곳에 내려준 경우
   - **대리기사 지각 불만**: 대리기사가 약속 시간에 도착하지 않은 경우
   - **대리기사 안전운전 미준수 불만**: 위험하게 운전한 경우
   - **대리 요금 청구 오류 불만**: 요금이 잘못 청구된 경우

### 2. AI 고객 음성 듣기

1. 시나리오 선택 후 AI 고객의 초기 메시지가 음성으로 재생됩니다
2. 이어폰이나 헤드셋을 착용하는 것을 권장합니다
3. 고객의 감정 상태를 파악하세요 (화난, 답답한, 슬픈 등)

### 3. 음성 녹음

1. "녹음 시작" 버튼 클릭
2. 마이크 권한 허용 (처음 사용 시)
3. 상담원으로서 고객에게 응대 내용을 말하세요
4. "녹음 종료" 버튼 클릭
5. 음성이 자동으로 텍스트로 변환됩니다

**팁**:
- 조용한 환경에서 녹음하세요
- 명확하고 천천히 말하세요
- 마이크와 적절한 거리를 유지하세요

### 4. 대화 진행

1. AI 고객이 상담원의 응대에 따라 반응합니다
2. 대화를 원하는 만큼 계속 진행할 수 있습니다
3. 고객의 감정이 변화하는 것을 확인하세요

### 5. 대화 종료 및 피드백 확인

1. "대화 종료" 버튼 클릭
2. AI가 응대 내용을 분석합니다 (약 10-20초 소요)
3. 피드백 화면에서 다음 정보를 확인하세요:
   - **공감 점수**: 고객의 감정을 이해하고 공감한 정도
   - **문제 해결 점수**: 문제를 효과적으로 해결한 정도
   - **전문성 점수**: 전문적이고 정확한 정보 제공 정도
   - **어조 점수**: 적절한 어조와 말투 사용 정도
   - **종합 점수**: 전체 평가 점수
   - **강점**: 잘한 부분
   - **개선점**: 개선이 필요한 부분
   - **상세 피드백**: 구체적인 피드백 내용

4. 점수가 사용자 통계에 반영되고 레벨업할 수 있습니다

## AI 이론 평가 사용법

### 1. 학습 자료 선택

1. "AI 이론 마스터" 모드 선택
2. 등록된 학습 자료 목록에서 원하는 자료 선택
3. "시험 보기" 버튼 클릭
4. AI가 문제를 출제합니다 (약 10-20초 소요)

### 2. 퀴즈 풀이

1. 문제가 순서대로 표시됩니다
2. 각 문제에 답변을 선택하세요:
   - **객관식 문제 (1-5번)**: 4개 보기 중 하나 선택
   - **O/X 문제 (6-10번)**: O(맞음) 또는 X(틀림) 선택
3. "다음 문제" 버튼으로 다음 문제로 이동
4. 마지막 문제에서 "결과 확인" 버튼 클릭

### 3. 결과 확인

1. 시험 결과 화면에서 다음 정보를 확인하세요:
   - **총점**: 100점 만점 기준 점수
   - **맞은 개수**: 정답 개수
   - **틀린 개수**: 오답 개수
2. 각 문제별 해설 확인:
   - 정답 및 사용자 답안
   - 상세 해설
3. 점수가 사용자 통계에 반영됩니다

## 게임화 기능

### 1. 레벨 시스템

- 점수를 획득하면 레벨이 올라갑니다
- 100점당 1레벨 증가
- 프로필 화면에서 현재 레벨 확인 가능

### 2. 배지 시스템

다양한 업적을 달성하면 배지를 획득할 수 있습니다:
- **첫 걸음**: 첫 번째 세션 완료
- **완벽주의자**: 100점 만점 달성
- **열정가**: 10개의 세션 완료
- **마스터**: 모든 시나리오 완료
- **연속 출석**: 3일 연속 사용
- **공감왕**: 공감 점수 평균 90점 이상
- **해결사**: 문제 해결 점수 평균 90점 이상
- **전문가**: 전문성 점수 평균 90점 이상

### 3. 리더보드

- 전체 사용자 중 자신의 순위 확인
- 총 점수, 레벨, 완료 세션 수 비교
- 프로필 화면에서 "리더보드" 버튼 클릭

## 관리자 기능

### 1. 관리자 권한 확인

관리자 계정으로 로그인하면 헤더에 "관리자" 버튼이 표시됩니다.

### 2. 사용자 권한 관리

1. "관리자" 버튼 클릭
2. "사용자 권한 관리" 탭 선택
3. 사용자 목록에서 원하는 사용자 선택
4. "관리자 지정" 또는 "관리자 해제" 버튼 클릭

### 3. 학습 자료 관리

1. "관리자" 버튼 클릭
2. "학습 자료 관리" 탭 선택
3. 학습 자료 업로드:
   - Zip 파일을 드래그 앤 드롭하거나 "파일 선택" 버튼 클릭
   - 파일 크기: 최대 1GB
   - 지원 형식: 텍스트 파일(.txt, .md 등), 파워포인트(.pptx)
4. 학습 자료 삭제:
   - 자료 목록에서 삭제할 자료의 삭제 버튼 클릭
   - 확인 후 삭제

## FAQ

### Q1: 음성 인식이 잘 안 됩니다
**A**: 다음을 확인해주세요:
- 조용한 환경에서 녹음하세요
- 마이크와 적절한 거리를 유지하세요 (10-20cm)
- 명확하고 천천히 말하세요
- 마이크 권한이 허용되어 있는지 확인하세요

### Q2: AI 고객의 음성이 들리지 않습니다
**A**: 다음을 확인해주세요:
- 브라우저의 음량 설정 확인
- 이어폰이나 헤드셋 연결 확인
- 브라우저가 오디오 재생을 허용하는지 확인

### Q3: 퀴즈 문제가 생성되지 않습니다
**A**: 다음을 확인해주세요:
- 학습 자료에 텍스트가 포함되어 있는지 확인
- 네트워크 연결 확인
- 잠시 후 다시 시도

### Q4: 배지를 획득하지 못했습니다
**A**: 배지는 특정 조건을 만족해야 획득할 수 있습니다:
- 첫 걸음: 첫 세션 완료
- 완벽주의자: 100점 만점 달성
- 열정가: 10개 세션 완료
- 기타 배지: 해당 조건 확인

### Q5: 관리자 권한이 없습니다
**A**: 관리자 권한은 다른 관리자가 부여해야 합니다. 관리자에게 문의하세요.

### Q6: 학습 자료를 업로드할 수 없습니다
**A**: 다음을 확인해주세요:
- 관리자 권한이 있는지 확인
- 파일 형식이 Zip인지 확인
- 파일 크기가 1GB 이하인지 확인
- 파일에 텍스트 파일이나 PPTX 파일이 포함되어 있는지 확인

### Q7: 점수가 반영되지 않습니다
**A**: 다음을 확인해주세요:
- 세션이 정상적으로 완료되었는지 확인
- 네트워크 연결 확인
- 페이지를 새로고침하여 통계 확인

## 문제 해결

### 음성 녹음 문제

**증상**: 녹음이 시작되지 않거나 중단됨

**해결 방법**:
1. 브라우저의 마이크 권한 확인
   - Chrome: 설정 > 개인정보 및 보안 > 사이트 설정 > 마이크
   - Firefox: 설정 > 개인정보 및 보안 > 권한 > 마이크
2. 다른 앱에서 마이크를 사용 중인지 확인
3. 브라우저를 재시작
4. 다른 브라우저에서 시도

### 음성 인식 오류

**증상**: 음성이 텍스트로 변환되지 않거나 잘못 변환됨

**해결 방법**:
1. 조용한 환경에서 다시 녹음
2. 명확하고 천천히 말하기
3. 마이크와의 거리 조정
4. 네트워크 연결 확인

### API 오류

**증상**: "API 할당량 초과" 또는 "서버 오류" 메시지 표시

**해결 방법**:
1. 잠시 후 다시 시도 (자동 재시도 기능)
2. 네트워크 연결 확인
3. Google Cloud Console에서 API 사용량 확인
4. 관리자에게 문의

### 로그인 문제

**증상**: 로그인이 되지 않음

**해결 방법**:
1. 이메일과 비밀번호 확인
2. 이메일 인증 완료 여부 확인
3. "비밀번호 재설정" 기능 사용
4. 브라우저 캐시 삭제 후 재시도

### 파일 업로드 문제

**증상**: 파일 업로드가 실패함

**해결 방법**:
1. 파일 크기 확인 (최대 1GB)
2. 파일 형식 확인 (Zip 파일만 가능)
3. 파일에 텍스트 파일이나 PPTX 파일이 포함되어 있는지 확인
4. 네트워크 연결 확인

### 피드백 생성 지연

**증상**: 피드백 생성에 시간이 오래 걸림

**해결 방법**:
1. 잠시 기다리기 (일반적으로 10-20초 소요)
2. 네트워크 연결 확인
3. 페이지를 새로고침하지 마세요 (처리 중입니다)
